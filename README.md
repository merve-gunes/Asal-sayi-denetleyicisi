# Asal Sayı Denetleyicisi

Canlı site: https://merve-gunes.github.io/Asal-sayi-denetleyicisi/

Basit, hızlı ve kullanıcı dostu bir JavaScript uygulaması.  
Girilen sayının asal olup olmadığını kontrol eder + ekstra özellikler sunar!

## Özellikler
- Girilen sayının **asal** olup olmadığını anında söyler  
- Sayıya kadar olan tüm **asal sayıları** listeler  
- Sayının kaçıncı asal sayı olduğunu gösterir (örnek: 7 → 4. asal sayı)  
- Temiz ve mobil uyumlu arayüz  
- Hiçbir kütüphane kullanılmadı, tamamen vanilla HTML + CSS + JavaScript

## Nasıl Kullanılır?
1. Sayı kutusuna pozitif bir tam sayı girin  
2. "Kontrol Et" butonuna tıklayın veya Enter tuşuna basın  
3. Sonuç hemen altında görünecek!

## Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox ile responsive tasarım)
- Vanilla JavaScript

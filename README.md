- Asal Sayı Denetleyicisi

Bu proje, kullanıcının girdiği sayının asal olup olmadığını kontrol eden basit ve kullanıcı dostu bir web uygulamasıdır.
Ayrıca girilen sayıya kadar olan asal sayı listesini çıkarma, sayının kaçıncı asal sayı olduğunu bulma ve görsel olarak sonuçları gösterme gibi ek özellikler içerir.


 -Özellikler

Girilen sayının asal olup olmadığını kontrol eder

Girilen sayıya kadar olan tüm asal sayıları listeler

Sayının kaçıncı asal sayı olduğunu gösterir

Modern ve kullanıcı dostu arayüz

HTML, CSS ve JavaScript ile oluşturulmuş basit bir web uygulaması
